# Ajout d'une page /welcome
Cette page sera la page de célébration une fois que le bébé sera né !

La page reprendra les informations sur les données du bébé, ainsi qu'une bannière affichant un message de bienvenue/félicitations.

Il faut ajouter les deux informations sur le poids et la taille avec les icones qui vont bien

Cette page affichera ensuite un podium (tu peux te baser sur la wireframe du fichier podium.excalidraw.png) qui permettra d'indiquer qui sont les gagnants des paris. 

Il doit y avoir des confettis à l'arrivée dans la page, et au clic sur les numéros du podium.

Tu peux reprendre les informations et les visuels sur les lots qui sont dans la page d'accueil. Ils ne seront pas restitués dans une modale, mais directement dans la page sous le podium.

Les avatars des gagnant doivent être utilisés, et animés en rebondissant pour célébrer la victoire. Leur prénom et nom doivent être affichés sur le podium en soulignés, et au clic on doit afficher un récap de leur pari dans un tooltip.

# Notes techniques
Utilise les MCP supabase, nuxtui, et les autres connectés quand tu en as besoin. 
Utilise les icones de lucid icon (tu as le mcp pour ça)
Utilise les composants de base de nuxtui, essaie de ne pas faire de html/css custom.
Utilise la wireframe Wireframes/podium.excalidraw.png pour comprendre le rendu voulu.
